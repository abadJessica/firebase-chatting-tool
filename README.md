# Firebase Realtime Database Chat

This activity demonstrates a simple **live chat application** using **Firebase Realtime Database**. Users can enter their name and a message, and the message will instantly appear in the chat for everyone viewing the page.

## Features
- Send messages with a name
- Messages update in real time
- Data stored in Firebase Realtime Database
- Simple HTML and JavaScript implementation

## Technologies Used
- HTML
- JavaScript (ES Modules)
- Firebase Realtime Database

## How It Works
1. User enters a **name** and **message**.
2. When the **Add Message** button is clicked, the message is pushed to Firebase.
3. Firebase updates the database.
4. The page listens for database changes and automatically displays new messages.
